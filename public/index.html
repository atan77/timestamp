FreeCodeCamp API: Timestamp Microservice

User stories:
    1) I can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: January 1, 2016)
    2) If it does, it returns both the Unix timestamp and the natural language form of that date.
    3) If it does not contain a date or Unix timestamp, it returns null for those properties.
Add your timestamp string to the url above and a JSON string will be returned in both unix time and natural time formats if the string is valid.

Example input time strings:

https://timestamp-atan77.c9users.io/1451865600
https://timestamp-atan77.c9users.io/2016-11-25

Example output:

{ current unix time: "1451884514", current natural time: "2016-01-04T05:15:14.099Z", unix time: "1480032000", natural time: "25-11-2016" }
